## How to Run

### Prerequisites
1. **API Must Be Running**
   ```bash
   # From project root
   python src/api/main.py
   ```
   API will start at http://localhost:8000

2. **Generate Sample Content** (Optional but Recommended)
   ```bash
   # Generate a few area guides to showcase the content generation feature
   python src/content/generate.py
   # Choose option 1 or 2 to generate content
   ```

### Launch the Frontend

#### Option 1: Simple HTTP Server (Python)
```bash
cd frontend
python -m http.server 8080
```
Then open: http://localhost:8080

#### Option 2: Live Server (VS Code Extension)
1. Install "Live Server" extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

#### Option 3: Direct File Opening
Simply open `frontend/index.html` in your web browser.
**Note**: Some features may not work due to CORS restrictions. Use Option 1 or 2 instead.

## Troubleshooting

### API Connection Failed
- **Issue**: "API Offline" indicator shows red
- **Fix**: Make sure the API is running at http://localhost:8000
- Run: `python src/api/main.py`

### No Sample Content
- **Issue**: Content section says "No content generated yet"
- **Fix**: Generate sample content first
- Run: `python src/content/generate.py` and generate 1-2 area guides

### CORS Errors
- **Issue**: Browser console shows CORS errors
- **Fix**: Don't open HTML file directly. Use HTTP server (Option 1 or 2 above)

### Content Generation Fails
- **Issue**: "Content generation failed" message
- **Fix**: Check that `ANTHROPIC_API_KEY` is set in `.env` file
- Verify API key is valid
